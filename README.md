# TinoHost Installation Package

[TinoHost](https://github.com/TinoHost-dev/TinoHost) is a modern, open-source Linux server management panel.

This project contains the TinoHost installation package, including the installation script and default configuration files.

## Issue Reporting

If you encounter any problems during use or have additional feedback, please submit a GitHub Issue to the [main TinoHost repository](https://github.com/TinoHost-dev/TinoHost/issues).
